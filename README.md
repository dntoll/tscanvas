# tscanvas


## dependencies


```
  sudo apt-get install npm
  curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
  sudo apt-get install -y nodejs
  sudo npm install -g typescript
  npm install --save-dev webpack
  npm install --save lodash
  chmod +x build.sh
```  

Run and build
```
  ./build.sh
```
